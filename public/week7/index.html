Move-and-shootathon

Using your own OTZ game server, implement a game client that allows players to move around the game world using their keyboard.

  • If you haven't already, in otz/ run npm install, then run: 
    node app.js
  • When the server is up, point your browser to http://localhost:3000. You should see the game world and an orange square representing you.
  • Now, edit otz/public/js/otz.js and create the keyboard event handlers to send player-move events to the game server. Refer to the docs for further instruction.
  • Once your player can move, create an additional handler to send the player-shoot event. Refer to the docs for further instruction.
  • Lastly, create a handler to send the player-turn event, so you can aim your gun without moving to another tile.
  • Note that the bare-bones client provides the jwerty keyboard event plugin for your convenience.

Spriteathon

Using the provided CSS classes and adding your own in otz/public/css/index.css, create sprites, textures and effects for your game client.

  • Create textures for the .grass and .rock world tiles.
  • Create a sprite to represent your character in .world .you.
  • Create a sprite (or multiple sprites) to represent other characters in .world .player.
  • Modify the .bullet styles to match your vision of a bullet.

Notifiathon

Whenever an interesting thing happens in the game, notify the user.

  • Use player-joined, player-killed, player-nick-changed and player-quit to notify the user when something interesting happens to players.
  • Use disconnect and reconnect to tell the user that they've lost the connection to the server and reconnected, respectively. This is also a fine oppurtunity to show a kill screen.
  • Allow the user to update thir nickname using a text box and sending the player-nick event.
  • Optionally, use the player-* events to create a leaderboard that keeps track of connected players and their kill/death ratio since your character connected.
