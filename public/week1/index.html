Git & Github

  • Git is a distributed version control system.
  • Basically, it lets you store every version of every file in any project.
  • GitHub is a web service that stores all of those Git repositories that store every version of every file in your projects.
  • If you haven't already, go sign up for a free GitHub account. Then, email your GitHub username to pwooley@aii.edu and I'll add you to the aipd-wdim493j GitHub organization.

Node.js as a REPL

  • In a terminal, run: 
    $ node
  • You're now in the Node REPL, try writing a little JavaScript: 
    > 2+2
4
> 2+2 !== 5
true
> var answer = 42;
undefined
> answer
42
> answer + 10
52
> {a: 1, b: answer}
{ a: 1, b: 42 }
> .exit

Node Package Manager

npm is a package manager for node. You can use it to install and publish your node programs. It manages dependencies and does other cool stuff." —npmjs.org

With npm...

  • You can find cool packages at search.npmjs.org.
  • You can install packages locally with something like: 
    $ npm install request
  • You can use the packages you install within a node program using code like: 
    var request = require('request');
request('http://www.google.com', function (error, response, body) {
  if (!error && response.statusCode == 200) {
    console.log(body) // Print the google web page.
  }
})

Screen Scrapathon

In pairs, you need to craft a Node.js app that finds the latest issue number of A List Apart. To do this, you'll use two Node.js packages available on npm: request and cheerio. Follow these steps and complete the implementation to achieve Hackathon Success:

  1. In week1/class, create a directory named ala.
  2. cd into ala and create a file named app.js.
  3. Run 
    $ npm install request cheerio
  4. Edit app.js and add the following: 
    var request = require('request')
    , cheerio = require('cheerio');
  5. Refer to the request docs on how to load in A List Apart's home page and cheerio's docs on how to screen scrape it to find the current issue number.
    6. At the end, you should be able to run: 
    $ node app.js
The latest ALA issue is number XXX
