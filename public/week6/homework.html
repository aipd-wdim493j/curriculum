Homework: Assignment 5, due Week 7

Socket Control

Using Socket.IO, craft a Node app that allows a user to control a uniquely-colored square with their keyboard, which is then displayed in the browser of all other users.

  • When a user connects, a div should be created and placed randomly on the screen with a random color.
  • Using the keyboard (up, down, left, right arrow keys; a,w,sd; etc.), the user should be able to move the div around the screen.
  • Using a Socket.IO the location and color of the user's div should be visualized on all other connected clients (users of the web app).
  • In the end, if your Node app has six users, then there will be six colored divs in each user's browser. One of the div will be under the control of the user.
  • For ease-of-use, consider making it obvious to the user which div is under their control (a thick border, the text "You" in the div, etc.).
